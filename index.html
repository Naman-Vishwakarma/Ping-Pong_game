Ping Pong Game

Press 'e' to Exit Game After, Game Started!

Single Player

Instructions

Press 'Enter' to start game

To move playing-pad use, key 'A' to move left & key 'D' to right.

You would be able to move both playing pads with same keys.
Which pad is to move, decided by the ball. If ball is in upper court the upper pad will move and same with lower pad.

Two Player

Instructions

Press 'Enter' to start game.

Player-1 : To move playing-pad use, key 'A' to move left & key 'D' to right.

Player-2 : To move playing-pad use, key 'J' to move left & key 'L' to right.
